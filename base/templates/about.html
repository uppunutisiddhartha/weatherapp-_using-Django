Project Overview

About the Project

This project is a simple weather application built using Django and OpenWeatherMap API. It allows users to search for any city and view the current weather conditions such as temperature, humidity, wind speed, and weather description.

Features:

  • Search weather by city name
  • Real-time data fetched from OpenWeather API
  • Displays temperature, humidity, pressure, wind speed, and weather description
  • Responsive design for better user experience
Technologies Used
  • Backend: Django (Python)
  • API: OpenWeatherMap API
  • Frontend: HTML, CSS, Bootstrap
